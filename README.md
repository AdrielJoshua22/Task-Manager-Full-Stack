# Task Manager

Hola, bienvenido al repositorio de mi gestor de tareas. 

Armé este proyecto personal para poner en práctica lo que vengo aprendiendo de Java, intentar aplicar principios de código limpio y, de paso, tener un programador de tareas por consola que sea rápido y útil para el día a día.

## Qué hace la aplicación
- Permite crear, leer, actualizar y eliminar tareas de forma sencilla.
- Se pueden agendar tareas asignándoles una fecha y hora específica.
- Toda la información se guarda de forma persistente.

## Stack Tecnológico
- Java (Lógica principal de la aplicación)
- MySQL (Base de datos)
- Consola / Terminal (Interfaz de usuario)

## Cómo probarlo en tu máquina
Si quieres clonar el proyecto para darle una mirada, los pasos son bastante simples:

1. Asegúrate de tener instalado Java en tu equipo.
2. Clona este repositorio desde tu terminal:
   git clone https://github.com/AdrielJoshua22/Task-Manager-Full-Stack.git
3. Abre el proyecto en tu IDE de preferencia (IntelliJ, Eclipse, etc.).
4. Configura las credenciales de conexión a tu base de datos MySQL local para que la app pueda guardar y leer los registros.
5. Ejecuta el archivo principal y listo.
